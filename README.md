# SmartPropoPlus
SmartPropoPlus - Turning your R/C transmitter into a Flight Simulator Joystick

## Features:
* Vista(SP2), Windows 7 and Windows 8.x
* 32 and 64 bit computers
* Supports most transmitters
* Detailed instructions in the site
